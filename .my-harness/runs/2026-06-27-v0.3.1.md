# base-portal v0.3.1 my-harness Run

日期：2026-06-27

## 当前范围

v0.3.1 是部署治理和权限 UI 范围收口补丁版。

本版本明确不做：

- 权限矩阵。
- 权限同步状态页。
- 审计入口。
- 权限绑定编辑、权限组选择器、权限点表格或批量操作 UI。
- 新 Product Design 视觉稿或 Pencil 原型。

## 目标

- 将仓库版本、部署默认值、安装脚本和升级脚本推进到 `0.3.1` / `v0.3.1`。
- 修复 `deploy/docker-compose.yml` 中 `BASE_PORTAL_VERSION` 和 `APP_VERSION` fallback 仍停在旧版本的问题。
- 保留 v0.3.0 的 SSO Demo 菜单、iframe refresh、接入包导入、IAM sync 和权限过滤能力。
- 发布 tag / GitHub Release `v0.3.1`。
- 部署到 92 服务器并读回 `.deploy/version`、容器 image、`/health`、`/ready` 和 `/version`。

## 关键决策

- 用户确认：本版本权限相关 UI 全部不做。
- Step 2/4/10 设计门禁跳过：本版本不新增用户可见 UI。
- Step 5 工程评审跳过：不改架构、数据库 schema、OAuth、权限 API 或 secret 边界。
