## 本地 Compose

启动命令使用独立项目名和 mock-auth 覆盖，不读取或覆盖真实 `deploy/.env`：

```bash
COMPOSE_PROJECT_NAME=base_portal_v020 \
POSTGRES_DATA_DIR=./data/postgres-v020 \
HOST_WEB_PORT=3103 \
PORTAL_ENABLE_MOCK_AUTH=true \
COOKIE_SECURE=false \
PORTAL_MOCK_USER_ID=mock-user-001 \
PORTAL_ADMIN_USER_IDS=mock-user-001 \
BASE_PORTAL_IMAGE=base-portal-local:v0.2.0 \
BASE_PORTAL_VERSION=v0.2.0 \
APP_VERSION=0.2.0 \
docker compose --env-file deploy/.env.example -f deploy/docker-compose.yml up -d --build
```
