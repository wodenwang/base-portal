## 生产升级命令

```bash

```

如果 release candidate 已经把远端 `.deploy/version` 写为 `v0.1.1`，但 `/version` 或容器环境仍需修正，可以显式使用：

```bash
./upgrade.sh --from v0.1.1 --to v0.1.1 --image base-portal-release:v0.1.1 --pull never --force
```
