# Base Portal v0.3.0 Pre-Landing Review

日期：2026-06-20

## Findings

未发现阻塞 v0.3.0 进入 ship / deploy 的代码问题。

## Review Notes

- SSO Demo 职责边界保持清晰：Base Portal 只提供菜单、权限过滤、iframe、新窗口恢复和 fallback，不传递 token、cookie、authorization code 或 secret。
- 前端 workspace runtime 已移除 `enterImmersiveTab` / `exitImmersiveTab` 和用户可见沉浸入口。
- `WorkspaceTab.openMode` 只保留运行态 `iframe`，历史 `NavigationMenu.openMode` 仍可接收 `immersive_iframe` 并在打开时降级。
- 当前 iframe refresh 使用 `refreshKey` 纯状态模型，右键菜单和三点菜单共用 `refreshTabFrame`。
- 后端 app package import 允许历史 `immersive_iframe`，apply 写入 `iframe`，dry-run/apply summary 只包含非敏感 warning。
- SSO Demo 接入包 URL 和权限点不包含 secret。
